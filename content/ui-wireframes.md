# UI 와이어프레임 문서

> 작성일: 2026-06-13 | 버전: 1.0  
> 해상도 기준: 1024 x 512px (Phaser 3 캔버스)

---

## 범례

```
[ 버튼 ]        → 클릭 가능한 버튼
[이미지]         → 이미지/스프라이트 영역
 █████          → 진행 바 (채워진 부분)
 ░░░░░          → 진행 바 (빈 부분)
 ~~~~           → 애니메이션 요소
 - - -          → 점선 구분선
 ===== / -----  → 구분선
(텍스트)         → 레이블/설명 텍스트
```
