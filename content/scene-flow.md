# 씬 전환 흐름 다이어그램

> 작성일: 2026-06-13 | 버전: 1.0

## 전체 씬 맵

```
┌─────────────────────────────────────────────────────────────┐
│                        씬 구성 전체                          │
│                                                             │
│  [TitleScene]                                               │
│       │                                                     │
│       ├── 새 게임 ──────────→ [CharacterSelectScene]        │
│       │                               │                    │
│       └── 이어하기 ──────┐            │ 캐릭터 선택         │
│                          │            ↓                    │
│                          └──→ [WorldMapScene] ◄────────┐   │
│                                     │    ▲             │   │
│                               A키   │    │ "다음" 버튼  │   │
│                             발견    │    │             │   │
│                                     ↓    │             │   │
│                               [PortScene]              │   │
│                               (교육 카드)              │   │
│                                                        │   │
│                         [기록장] ──→ [LogbookScene] ───┘   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 씬 1: TitleScene

### 진입 조건
- 앱 최초 실행 시 자동 진입
- 다른 씬 없음 — 항상 시작점

### 화면 구성 요소
```
+--------------------------------------------------+
|                                                  |
|            세계 탐험! (큰 타이틀)                |
|         World Explorer                           |
|                                                  |
|          [배 + 파도 애니메이션]                  |
|                                                  |
|              [ 새 게임 시작 ]                    |
|              [ 이어하기    ]  ← 저장 데이터 없으면 비활성
|                                                  |
|         [처음부터 다시] (작은 텍스트)            |
+--------------------------------------------------+
```

### 화면 구성 요소 상세
| 요소 | 설명 |
|------|------|
| 배경 | 파도 애니메이션이 있는 바다 배경 (타일링 스프라이트) |
| 타이틀 로고 | "세계 탐험!" 큰 텍스트 + "World Explorer" 부제목 |
| 배 애니메이션 | 작은 배가 좌우로 느리게 흔들리는 idle 애니메이션 |
| 새 게임 버튼 | 항상 활성화 |
| 이어하기 버튼 | localStorage에 저장 데이터 있을 때만 활성화 (없으면 흐리게) |
| 처음부터 다시 | 저장 초기화 버튼, 최하단 작은 텍스트 |

### 사용자 인터랙션
| 인터랙션 | 결과 |
|----------|------|
| "새 게임 시작" 클릭 | CharacterSelectScene으로 전환 |
| "이어하기" 클릭 | WorldMapScene으로 직행 (저장된 위치에서 시작) |
| "처음부터 다시" 클릭 | 확인 다이얼로그 → 확인 시 localStorage 초기화 후 TitleScene 재로드 |

### 다음 씬 전환 트리거
- 새 게임 버튼 클릭 → CharacterSelectScene (페이드 아웃/인 0.4s)
- 이어하기 버튼 클릭 → WorldMapScene (페이드 아웃/인 0.6s)

## 씬 2: CharacterSelectScene

### 진입 조건
- TitleScene에서 "새 게임 시작" 버튼 클릭 시

### 화면 구성 요소
```
+--------------------------------------------------+
|                                                  |
|       누구와 함께 떠날까?                        |
|                                                  |
|  +-------------------+ +-------------------+    |
|  |                   | |                   |    |
|  |  [준 일러스트]    | |  [아라 일러스트]  |    |
|  |                   | |                   |    |
|  |  항해사 준        | |  탐험가 아라      |    |
|  |  Jun the Navigator| |  Ara the Explorer |    |
|  |                   | |                   |    |
|  |  파란 모자 착용   | |  빨간 두건 착용   |    |
|  |  밝고 호기심 많음 | |  용감하고 지식욕  |    |
|  |                   | |                   |    |
|  |  [선택하기]       | |  [선택하기]       |    |
|  +-------------------+ +-------------------+    |
|                                                  |
|     "어떤 탐험가가 될지 골라봐!" (설명 문구)     |
+--------------------------------------------------+
```

### 화면 구성 요소 상세
| 요소 | 설명 |
|------|------|
| 제목 | "누구와 함께 떠날까?" 상단 중앙 |
| 준 카드 | 좌측, 파란 테마, 캐릭터 전신 일러스트 + 이름/설명 |
| 아라 카드 | 우측, 빨간 테마, 캐릭터 전신 일러스트 + 이름/설명 |
| 선택하기 버튼 | 각 카드 하단, 대형 버튼 |
| 배경 | 옅은 하늘+바다 배경 |

### 사용자 인터랙션
| 인터랙션 | 결과 |
|----------|------|
| 준 카드 hover | 카드 위로 떠오름 (y -8px), 테두리 파란 글로우 |
| 아라 카드 hover | 카드 위로 떠오름 (y -8px), 테두리 빨간 글로우 |
| "선택하기" 클릭 | 확인 다이얼로그: "[이름]와(과) 함께 떠날게요! 출항!" |
| 출항 버튼 클릭 | WorldMapScene으로 전환 |

### 다음 씬 전환 트리거
- 캐릭터 선택 + 출항 확인 → WorldMapScene (페이드 아웃/인 0.6s)
  - 전환 시 선택 캐릭터 정보 localStorage에 저장

## 씬 3: WorldMapScene

### 진입 조건
- CharacterSelectScene에서 캐릭터 선택 완료 시 (새 게임)
- TitleScene에서 "이어하기" 클릭 시 (저장 데이터 불러오기)
- PortScene에서 "다음 탐험 계속하기" 클릭 시
- LogbookScene에서 "뒤로" 클릭 시

### 화면 구성 요소
```
+--------------------------------------------------+
| [배 아이콘] 준의 대항해 | 🌍 발견: 12/50 | 🏆 탐험가 | [📖] |
+--------------------------------------------------+
|                                                  |
|      [세계 지도 1024x472]                        |
|                                                  |
|    • • • •  (노란 점: 미발견 항구)               |
|                                                  |
|        ⚓ "A키로 발견!" (근접 시만 표시)         |
|                                                  |
|       ⛵ (플레이어 배 아이콘, 조종 가능)         |
|                                                  |
+--------------------------------------------------+
```

### 화면 구성 요소 상세
| 요소 | 설명 |
|------|------|
| 상단 HUD 바 | 40px 높이, 어두운 반투명 배경 |
| 캐릭터명 + 배 | HUD 좌측, 선택 캐릭터 소형 아이콘 + "[이름]의 대항해" |
| 발견 카운터 | HUD 중앙, 큰 숫자 강조, "🌍 발견: 12/50" |
| 현재 칭호 | HUD 중앙우, 칭호 배지 아이콘 + 텍스트 |
| 기록장 버튼 | HUD 우측, 📖 아이콘 + "기록장" 텍스트, 클릭 가능 버튼 |
| 세계 지도 | 전체 배경, 1024x472px (HUD 40px 제외) |
| 노란 점 | 미발견 항구들, 8px 원, alpha 0.6 |
| 닻 아이콘 | 근접 항구에만 표시, 펄스 애니메이션 |
| 배 아이콘 | 플레이어 조종 배, 32x32px, 이동 방향 회전 |
| 방문 깃발 | 발견한 항구, 대륙별 색상 16px 깃발 |

### 사용자 인터랙션
| 인터랙션 | 결과 |
|----------|------|
| 방향키 / WASD 누르기 | 배 이동 (120px/s, 경계 처리) |
| 항구 반경 60px 진입 | 닻 아이콘 + "A키로 발견!" 힌트 표시 |
| A키 또는 스페이스 (항구 근접 시) | 발견 연출 → PortScene 전환 |
| 기록장 버튼 클릭 | LogbookScene으로 전환 |

### 다음 씬 전환 트리거
- 항구 근접 + A키 → PortScene (슬라이드 업 0.4s)
- 기록장 버튼 → LogbookScene (페이드 아웃/인 0.3s)
- 칭호 임계값 달성 시 → 칭호 승급 오버레이 (WorldMapScene 위에 레이어)

## 씬 4: PortScene

### 진입 조건
- WorldMapScene에서 항구 근접 후 A키 / 스페이스 입력 시
- 발견 연출 완료 후 자동 전환

### 화면 구성 요소
```
+--------------------------------------------------+
|  [WorldMap 배경, 반투명 다크 오버레이]           |
|                                                  |
|  +------------------------------------------+   |
|  |  ✕                                       |   |
|  |  [랜드마크 일러스트 160x120px]            |   |
|  |                                           |   |
|  |  도쿄 / Tokyo                  🇯🇵        |   |
|  |  일본 수도 · 아시아                       |   |
|  |                                           |   |
|  |  ┌───────────────────────────────┐       |   |
|  |  │ 특산품                        │       |   |
|  |  │ 🍣 초밥 / Sushi              │       |   |
|  |  │ 🌸 벚꽃 / Cherry Blossom    │       |   |
|  |  │ 📺 전자제품 / Electronics   │       |   |
|  |  └───────────────────────────────┘       |   |
|  |                                           |   |
|  |  💡 알고 있었나요?                        |   |
|  |  도쿄는 세계에서 인구가 가장 많은...      |   |
|  |                                           |   |
|  |      [ 다음 탐험 계속하기 → ]             |   |
|  +------------------------------------------+   |
|                                                  |
+--------------------------------------------------+
```

### 화면 구성 요소 상세
| 요소 | 설명 |
|------|------|
| 배경 오버레이 | WorldMap 위에 반투명 다크 (alpha 0.5), 카드 뒤 배경 |
| 카드 컨테이너 | 중앙 정렬, 최대 480px 너비, 둥근 모서리(20px), 흰 배경 |
| ✕ 닫기 | 카드 우상단, 언제든 닫을 수 있음 |
| 랜드마크 이미지 | 160x120px, 카드 상단 중앙 |
| 도시명 | 한국어 대형(28px) + 영어 소형(16px), 국기 이모지 |
| 위치 설명 | "[국가] [대륙]", 14px 회색 텍스트 |
| 특산품 박스 | 둥근 테두리 박스, 3개 항목, 이모지 + 한국어/영어 |
| 재미있는 사실 | 💡 아이콘 + 텍스트, 타이핑 효과로 등장 |
| 다음 버튼 | 카드 최하단, 60% 너비, 주황 (#FF8C00) |

### 사용자 인터랙션
| 인터랙션 | 결과 |
|----------|------|
| ✕ 버튼 클릭 | 카드 닫기 → WorldMapScene 복귀 (도장은 이미 획득) |
| "다음 탐험 계속하기" 클릭 | 저장 → 카드 닫기 → WorldMapScene 복귀 |
| 카드 외부 클릭 | 아무 반응 없음 (실수 닫기 방지) |

### 다음 씬 전환 트리거
- "다음 탐험 계속하기" 또는 ✕ 클릭 → WorldMapScene (슬라이드 다운 0.3s)

## 씬 5: LogbookScene

### 진입 조건
- WorldMapScene에서 "기록장" 버튼 클릭 시

### 화면 구성 요소
```
+--------------------------------------------------+
| ← 뒤로   📖 나의 탐험 기록장   🏆 탐험가 12/50  |
+--------------------------------------------------+
| [탭: 방문 도시] [탭: 세계 지도]                  |
|                                                  |
| ─── 방문 도시 탭 ───                            |
|                                                  |
| +----+ +----+ +----+ +----+ +----+              |
| |이미지| |이미지| |이미지| |이미지| | ??? |       |
| |도쿄 | |파리 | |뉴욕 | |시드니| |     |       |
| +----+ +----+ +----+ +----+ +----+              |
|                                                  |
| ─── 세계 지도 탭 ───                            |
|                                                  |
| [800x400 세계 지도, 방문 지역 컬러]             |
|                                                  |
| 아시아   ████████░░ 8/15                        |
| 유럽     ████░░░░░░ 4/12                        |
| 아메리카 ░░░░░░░░░░ 0/10                        |
+--------------------------------------------------+
```

### 화면 구성 요소 상세
| 요소 | 설명 |
|------|------|
| 상단 바 | "뒤로" 버튼(좌) + 제목 중앙 + 진행 현황(우) |
| 탭 바 | "방문 도시" / "세계 지도" 두 탭 |
| 도시 그리드 | 5열 그리드, 100x130px 카드, 스크롤 가능 |
| 도시 카드 | 썸네일 70x70 + 도시명 한/영 |
| 미발견 카드 | 회색 실루엣 + "?" 마크 |
| 세계 지도 | 800x400, 방문 지역 컬러 오버레이 |
| 대륙별 진행 바 | 각 대륙 방문 수 / 전체 수 |

### 사용자 인터랙션
| 인터랙션 | 결과 |
|----------|------|
| "뒤로" 버튼 클릭 | WorldMapScene으로 복귀 |
| "방문 도시" 탭 클릭 | 도시 그리드 뷰 표시 |
| "세계 지도" 탭 클릭 | 세계 지도 오버레이 뷰 표시 |
| 방문 도시 카드 클릭 | 해당 도시 간단 정보 팝업 (PortScene 축약판) |
| 미발견 카드 클릭 | 아무 반응 없음 |
| 그리드 스크롤 | 마우스 휠 또는 터치 드래그로 스크롤 |

### 다음 씬 전환 트리거
- "뒤로" 버튼 → WorldMapScene (페이드 아웃/인 0.3s)

## 씬 전환 타이밍 요약표

| 출발 씬 | 도착 씬 | 트리거 | 전환 방식 | 시간 |
|---------|---------|--------|----------|------|
| TitleScene | CharacterSelectScene | 새 게임 버튼 | 페이드 아웃/인 | 0.4s |
| TitleScene | WorldMapScene | 이어하기 버튼 | 페이드 아웃/인 | 0.6s |
| CharacterSelectScene | WorldMapScene | 캐릭터 선택+확인 | 페이드 아웃/인 | 0.6s |
| WorldMapScene | PortScene | 항구 근접+A키 | 슬라이드 업 | 0.4s |
| PortScene | WorldMapScene | 다음/닫기 버튼 | 슬라이드 다운 | 0.3s |
| WorldMapScene | LogbookScene | 기록장 버튼 | 페이드 아웃/인 | 0.3s |
| LogbookScene | WorldMapScene | 뒤로 버튼 | 페이드 아웃/인 | 0.3s |

## Phaser 3 씬 구현 참고

```typescript
// 씬 키 상수
const SCENES = {
  TITLE: 'TitleScene',
  CHARACTER_SELECT: 'CharacterSelectScene',
  WORLD_MAP: 'WorldMapScene',
  PORT: 'PortScene',
  LOGBOOK: 'LogbookScene',
} as const;

// 전환 예시: PortScene으로 슬라이드 업
this.scene.transition({
  target: SCENES.PORT,
  duration: 400,
  moveBelow: true,
  onUpdate: (progress: number) => {
    portScene.cameras.main.y = (1 - progress) * 600;
  },
  data: { portId: 'tokyo' }
});
```
