# 대항해시대2 리서치 — 게임 개발 참고 자료

> 작성일: 2026-06-13 | 담당: writer + designer 에이전트 리서치

---

## 1. 게임 개요

**대항해시대2: New Horizons** (大航海時代 II, 1994, KOEI)  
탐험 + 무역 + 전투 + 항해가 결합된 전략 시뮬레이션 RPG. 15~16세기 대항해시대를 배경으로 세계 각지를 탐험하는 게임.

---

## 2. 핵심 게임플레이 루프

```
항구 출발 → 항로 탐색 → 무역/탐험 → 새 항구 발견 → 명성 획득 → 스토리 진행
```

**5대 핵심 시스템**:
1. 탐험 & 발견 — 새 항구, 지형, 문화 발견
2. 무역 — 항구 간 물품 매매
3. 선원 관리 — 전문 선원 고용·육성
4. 해전 — 해적 전투
5. 캐릭터 성장 — 무역/해적/모험 명성 누적

---

## 3. 세계 지도 구조

**13개 해역**으로 구성된 실제 세계 지도:
- 지중해 (Venice, Genoa, Istanbul, Athens, Naples)
- 대서양 연안 (Lisbon, London, Amsterdam)
- 카리브해 (Havana, Santo Domingo, Acapulco)
- 홍해·페르시아만
- 인도양 (Calicut, Kolkata)
- 동남아시아
- 북해·북유럽
- 흑해
- 태평양 항로

**총 100개 이상 항구**를 실제 좌표 위치에 배치.

---

## 4. 항구 시스템

### 항구 기능 (각 항구에서 할 수 있는 것)
- **교역소**: 물품 매매
- **선술집**: 선원 고용, 소문 입수, 퀘스트
- **조선소**: 배 구입/수리/개조
- **항무소**: 정보, 임무, 스토리 진행
- **마을 탐색**: 발견물 획득

### 문화 권역 분류
같은 문화권 항구는 비슷한 물품을 갖고 있고 경제적으로 연결됨.

---

## 5. 무역 물품 시스템

**46종 무역 물품** / **10개 카테고리** / **13개 지역 시장**

### 물품 카테고리
| 카테고리 | 예시 |
|---------|------|
| 귀금속 | 금, 은 |
| 직물 | 비단, 면화, 모직 |
| 향신료 | 정향, 후추, 육두구 |
| 예술·문화 | 미술품, 카펫 |
| 식품·농업 | 곡물, 설탕 |
| 원자재 | 목재, 철, 옥 |
| 사치품 | 향료, 향수 |

### 수익성 높은 무역 루트 예시
- 아테네 → 이스탄불: 미술품 ↔ 카펫
- 제노바 → 마데이라: 은 ↔ 금

### 경제 메커닉
- **가격 인플레이션**: 동일 항구에서 반복 판매 시 가격 하락
- **수요 변동**: 플레이어 활동에 따라 시장 가격 조정
- **지역 특산품**: 특정 물품은 특정 지역에서만 구매 가능

---

## 6. 발견 시스템

### 발견 종류
1. **지리적 발견**: 새 항구, 섬, 지형
2. **문화적 발견**: 마을, 유적, 기념물
3. **자연적 발견**: 식물, 동물, 자연 현상

### 발견 과정
- 알려진 항로 외 지역 탐험
- 마을 주민 친밀도 높이기 (Entertain 명령)
- 원격지 발견일수록 높은 명성 보상

### 발견 보상
- 명성(Fame) 획득
- 세계지도 공개
- 새 무역 루트 개방
- 스토리 진행 언락

---

## 7. 캐릭터 시스템

### 플레이어블 캐릭터 (6명)
1. **조앙 프랑코** (포르투갈) — 균형잡힌 탐험가, 아틀란티스 탐색
2. **카탈리나 에란초** (스페인) — 해적 전환, 복수 스토리
3. **오토 베인스** (영국) — 전투 특화, 스페인 함대 격파
4. **에른스트 폰 보어** (네덜란드) — 지도 제작 특화, 세계 완전 탐험
5. **피에트로 콘티** (이탈리아) — 보물 사냥꾼
6. (6번째 캐릭터)

### 선원 역할 분류
- 경리사무원 (재정 관리)
- 지도 제작자 (발견 능력 강화)
- 선장 (추가 배 지휘)
- 항해사 (항해 성능 향상)
- 전투원 (전투 능력)
- 탐험가 (발견 확률 증가)

---

## 8. 항해 메커닉

### 바람·해류 시스템
- 배는 **풍향**을 고려해야 함
- **해류**가 속도에 영향
- 노(노젓기)로 바람 무관 이동 가능 (페널티 있음)

### 선박 종류
- 소형: 빠른 탐험용
- 중형: 무역/전투 균형
- 대형: 대용량 화물
- 특수 능력: 쇄빙, 내파, 천수 항해 등

---

## 9. 우리 게임에 적용할 요소

### 채택할 요소
| 원작 요소 | 우리 적용 |
|---------|---------|
| 세계 지도 + 항구 네트워크 | 50개 주요 항구 |
| 발견 시스템 (Discovery) | 발견 도장 + 기록장 |
| 항구마다 특산품 | 도시당 특산품 3개 |
| 진행 명성 시스템 | 칭호 4단계 |
| 6명 캐릭터 | 2명으로 단순화 (준/아라) |

### 제거할 요소 (4학년 교육 목적 불필요)
- 복잡한 무역 경제 (가격 변동, 이윤 계산)
- 해전 시스템
- 선원 고용/관리
- 바람/해류 시뮬레이션

### 교육적 강화 포인트
- 각 도시 발견 시 교육 카드 → 지리/문화/특산품 학습
- 한국어 + 영어 병기로 이중 언어 학습
- 재미있는 사실로 흥미 유발
- 발견 기록장으로 학습 진도 시각화

## 10. 성공 요인 분석 (게임 디자인 패턴)

| 패턴 | 내용 | 우리 게임 적용 |
|------|------|--------------|
| 의미있는 선택 | 여러 캐릭터, 여러 루트 | 캐릭터 선택, 탐험 순서 자유 |
| 실세계 기반 | 실제 지명, 실제 특산품 | 정확한 지리·문화 정보 |
| 발견 기반 학습 | 팝업으로 교육 정보 전달 | 발견 카드 = 교육 순간 |
| 다중 목표 | 단기/중기/장기 목표 병존 | 도시 발견 / 칭호 / 전체 완성 |
| 내재적 동기 | 명성·발견이 자연스러운 보상 | 발견 도장, 칭호가 보상 |
