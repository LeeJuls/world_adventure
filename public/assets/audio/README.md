# BGM

Drop your legally-obtained main theme here, named exactly:

```
main-theme.mp3
```

→ `public/assets/audio/main-theme.mp3`

The game loads it automatically. Until the file exists, the game runs fine with no
music (a single console warning is logged). Music starts on the first click on the
title screen (browser autoplay policy), loops, and can be muted via the 🔊 button in
the bottom-left corner. Volume/mute preference is saved in localStorage.

> Use a track you have the rights to. Do not commit copyrighted audio to a public repo.
